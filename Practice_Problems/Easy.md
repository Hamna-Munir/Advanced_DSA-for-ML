# Easy Practice Problems

## Overview
This section contains **beginner-friendly problems** designed to strengthen fundamental understanding of **Data Structures and Algorithms (DSA)**.

These problems focus on building confidence with basic algorithmic thinking, simple data manipulation, and introductory problem-solving patterns.

They are ideal for learners who are starting their DSA journey.

---

## Topics Covered

- Arrays
- Strings
- Basic loops
- Conditional logic
- Simple searching
- Basic recursion

---

## Practice Problems

### 1. Two Sum
Given an array of integers and a target value, return the indices of the two numbers that add up to the target.

Example:

Input:

nums = [2,7,11,15]
target = 9

nums = [2,7,11,15]
target = 9


Output:

[0,1]


---

### 2. Reverse a String
Write a function that reverses a string.

Example:

Input:

"hello"


Output:

"olleh"


---

### 3. Find Maximum Element in Array
Given an array of numbers, find the largest element.

Example:

Input:

[3, 7, 2, 9, 4]


Output:

9


---

### 4. Palindrome Check
Check whether a string is a palindrome.

Example:

Input:

"madam"


Output:

True


---

### 5. Count Even Numbers
Given an array of integers, count how many numbers are even.

Example:

Input:

[1,2,3,4,5,6]


Output:

3


---

## Goal
These problems help learners develop **basic programming logic and algorithmic thinking**.
