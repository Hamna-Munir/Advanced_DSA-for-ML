# Graphs — Data Structures & Algorithms

## Overview
This module introduces **Graph data structures and algorithms**, which are essential for solving problems involving **relationships, networks, and connectivity**.

Graphs are a core concept in:
- Machine Learning
- Deep Learning (Graph Neural Networks)
- Recommendation systems
- Social network analysis
- Knowledge graphs
- Routing and optimization problems

This section focuses on **both theoretical understanding and practical implementations**, with a strong emphasis on ML applications.

---

## Learning Objectives
By completing this module, you will be able to:
- Understand different graph representations
- Traverse graphs efficiently using BFS and DFS
- Solve shortest path problems
- Analyze graph algorithms using time and space complexity
- Connect graph algorithms to real-world ML systems

Graphs are everywhere in ML:
